How to install BuildBot
=======================

1. Windows: follow instructions at http://trac.buildbot.net/wiki/RunningBuildbotOnWindows
   Other OSes: similar steps should be followed

2. Master only: update the list of authorized users in authz_cfg

3. Master only: modify master.cfg l.37 with the path to hg binary

4. Use code/build/buildbot/master.cfg to generate a local buildbot slave
   instructions at http://buildbot.net/buildbot/docs/current/tutorial/firstrun.html (buildslave create-slave)

5. Modify buildbot.*.egg/buildbot/changes/hgpoller.py to enforce UTF-8 comments
   (otherwise it'll choke when non ASCII-characters have been used in the svn history)
   line 86, in _getRevDetails, args = ['log', '--encoding', self.encoding, '-r', rev, os.linesep.join((

6. Windows: add DevEnv (or VCExpress) to the PATH environment variable

7. Starts BuildBot with buildbot start <master> or buildslave start <slave>
